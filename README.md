# Meteo-Parapente – Backup Favoris (version test)

Extension navigateur permettant d’exporter de trier et d’importer les favoris du site meteo-parapente.com.

Version hors Web Store, destinée à des tests internes.

## Installation (Chrome / Chromium)

* Télécharger [l’archive ZIP](https://github.com/devloben/favoris_meteo-parapente/archive/refs/heads/main.zip)

* Décompresser le dossier

* Ouvrir Chrome et aller sur : Fenêtre/Extensions ou saisir ```chrome://extensions``` dans la barre d'url du navigateur

* Activer _Mode développeur_ (en haut à droite de la page)

* Cliquer sur _Charger l’extension non empaquetée_

* Sélectionner le dossier de l’extension

* La carte de l'extension apparaît dans la liste

* Cliquer sur le bouton _Détails_ de la carte et activer _Épingler à la barre d'outils_. L’icône restera visible en permanence.

## Utilisation

* Rendez-vous sur https://meteo-parapente.com 
* Cliquer sur l'icone Météo Parapente dans la barre des extensions et suivez les instructions.
