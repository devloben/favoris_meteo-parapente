# Meteo-Parapente – Backup Favoris (version test)

Extension navigateur permettant d’exporter de trier et d’importer les favoris du site meteo-parapente.com.

Version hors Web Store, destinée à des tests internes.

## Installation (Chrome / Chromium)

* Télécharger le dossier de l’extension (ou l’archive ZIP)

* Décompresser le dossier

* Ouvrir Chrome et aller sur : Fenêtre/Extensions ou saisissez ```chrome://extensions``` dans la barre du navigateur

* Activer Mode développeur (en haut à droite de la page)

* Cliquer sur Charger l’extension non empaquetée

* Sélectionner le dossier de l’extension

* La carte de l'extension apparaît dans la liste

* Cliquer sur le bouton **Détails** de la carte et activer **Épingler à la barre d'outils**. L’icône restera visible en permanence.

## Utilisation

Rendez-vous sur https://meteo-parapente.com et suivez les instructions.
